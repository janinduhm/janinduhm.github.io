**Period:** Oct 2020 – Jun 2021 | B.Sc. Thesis

## Overview

A cost-effective indoor positioning system for accurate smartphone position estimation using **Wi-Fi RSSIs** and **accelerometer data**, developed as the B.Sc. final year project.

## Approach

- Shallow neural network classifies user location from preprocessed Wi-Fi signals
- Accelerometer-derived motion features improve input stability and compensate for user movement
- System designed to be versatile across different indoor environments without expensive infrastructure
