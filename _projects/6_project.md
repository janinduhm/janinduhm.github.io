**Period:** Mar 2020 – Jun 2021

## Overview

A cost-effective solution to convert conventional electric appliances into smart devices, integrating **Wi-Fi** technology within the smart home concept — without replacing existing appliances.

## Features

- Control existing home appliances via smartphone over Wi-Fi
- Gesture-based control interface
- Low-cost hardware based on NodeMCU ESP8266

## Tools

NodeMCU (ESP8266), Android Studio
