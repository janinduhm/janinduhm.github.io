**Period:** Mar 2020 – Jun 2021 | Mobile Application Development Course

## Overview

A lightweight Android note-taking application with persistent local storage, built as a semester project for the Mobile Application Development course.

## Features

- Full CRUD functionality (Create, Read, Update, Delete)
- Persistent local storage via SQLite
- Automatic timestamps on notes
- Material Design UI with RecyclerView and floating action button
- Empty state indicator when no notes exist

## Tools

Java, SQLite (SQLiteOpenHelper), Android Studio
